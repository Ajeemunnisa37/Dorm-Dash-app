# Dorm-Dash 🚀  
### Hyper-Local Campus Ecosystem

Dorm-Dash is a full-stack, mobile-first web application built to simplify everyday student life within a college campus.

It creates a **trusted, hyper-local ecosystem** where students can help each other through micro-errands, resource sharing, and skill exchange — all within a verified campus network.

---

## 🌟 Features

### 🛵 Micro-Errands (Dash)
- Post and accept small tasks within campus  
- Example: Food pickup, item delivery, quick help  
- Real-time request updates  

### 📦 Borrow-Bin
- Share and borrow items like:
  - Textbooks  
  - Calculators  
  - Lab equipment  
- Request → Approval → Borrow flow  

### 🎯 Skill-Swap
- Connect with peers to exchange skills  
- Example: Programming ↔ Physics  
- Match → Request → Accept → Chat  

---

## 🔐 Security & Trust

- Student-only access via college ID verification  
- Closed campus ecosystem  
- Safe peer-to-peer interactions  

---

## 📍 Smart Features

- Geofencing (500m radius filtering)  
- Real-time updates using WebSockets  
- Schedule-aware notifications  
- Live chat system

**Backend:**
- Node.js + Express  

**Database:**
- MongoDB  

**Other:**
- WebSockets (real-time features)

---

## 🔄 How It Works

1. User posts a request  
2. Nearby students can view it  
3. Another user accepts the request  
4. Both users get connected via chat  
5. Task gets completed  

---

## 🎯 Goal

Dorm-Dash aims to transform small campus inconveniences into a **collaborative digital ecosystem**, making student life more efficient, connected, and resourceful.
